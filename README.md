# AI Research Assistant (RAG + Cloud Ready)

This project is a **ChatGPT-style AI assistant** built using **Retrieval-Augmented Generation (RAG)**.

The system reads documents (PDF/text), converts them into embeddings, stores them in a vector database (FAISS), and retrieves relevant knowledge to answer user queries.

This project demonstrates **AI + Cloud architecture used in modern AI systems**.

---

## Project Architecture

User Question
↓
Embedding Model (SentenceTransformer)
↓
Vector Database (FAISS)
↓
Document Retrieval
↓
AI Response

---

## Technologies Used

- Python
- LangChain
- SentenceTransformers
- FAISS Vector Database
- Streamlit (UI)
